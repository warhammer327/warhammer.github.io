Data & Access Management
  • Core Functionality: CRUD operations for companies
  • Access Control: CanCancan gem
    • Role-based access control
    • User permission management

The project involves creating a Ruby on Rails API application using Ruby 3.0.0 and Rails 7, with a Postgres database and Postman for testing. The focus is on building a lean Rails API project. It includes setting up Devise for authentication and using Devise-JWT for handling JSON Web Tokens. Additionally, the project implements rate limiting using Rack-Attack to control API request rates. CRUD operations are built for managing companies, and access control is implemented based on user roles such as manager, admin, customer using the Cancancan gem.
