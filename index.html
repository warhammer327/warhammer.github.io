Projects

BeeSmart

Android Game

I developed this android game using Unity and C#. Aside from assets like fonts,music,etc I wrote all of the code, mechanisms, logics, and algorithms myself. It has Microsoft's playfab api in the backend that handles the leaderboard. There are two leaders board on is weekly updated across the globe another is the all time leaderboard.
The popular game 'Hangaroo' inspired this creation. We have largely concentrated on Bangladesh-related subjects. We attempted to cover everything from sports to historical landmarks to political history in this game. I had to use free materials such as fonts, music, and other works created by many artists.
